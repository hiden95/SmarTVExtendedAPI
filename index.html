Smart TV Extended API (HTML5 Based Smart TV Platform)

This document introduces Smart TV Extended APIs. These APIs are defined in "HTML5 Based Smart TV Platform"(TTAK.KO-07.0111/R1) to control Smart TV specific features like channel, program, device and so on. The specification has been standardized for common platform of Smart TV based on W3C web runtime environment to make an application running on various devices with the same user experiences. Smart TV Extended APIs consist of application interfaces, broadcast interfaces, device interface, multiscreen interfaces and DRM interfaces. These interfaces can be accessed as attributes of window object that is used globally in web environment.
